# Mi primer modelo

Este ejercicio forma parte del bootcamp de Factoria F5 y tiene como objetivo modelar el concepto de una persona mediante una clase en Java.

La persona debe incluir los siguientes atributos:

- nombre
- apellido
- número de documento de identidad
- año de nacimiento
- edad, calculada en función del año de nacimiento

La clase debe contar con un constructor que inicialice correctamente sus atributos. Además, la edad debe calcularse mediante un método, no de forma manual.

> Nota: no es necesario implementar una salida por terminal ni desarrollar una aplicación de consola.

## Requisitos

- Modelar la entidad Persona con sus atributos correspondientes.
- Implementar un constructor para inicializar los valores de la clase.
- Crear las pruebas pertinentes para asegurar el correcto funcionamiento del modelo.
- Alcanzar un coverage mínimo del 70%.

## Entregables

- Diagrama de clases (enlace o captura de pantalla).
- Captura de pantalla de la sección de Testing de Visual Studio Code donde se observe la cobertura de las pruebas.
